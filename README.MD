# Chris's DotFiles

## Description

Setup dotfiles `.zshrc` from github.

```bash

# clone dotfiles repo locally
cd ~
mkdir -p ~/code/p
cd ~/code/p
git clone https://github.com/ChrisTowles/dotfiles.git
cd dotFiles


# make symbolic link for .zshrc
cd ~/
ln -s /home/ctowles/code/dotfiles/.zshrc .zshrc
```
